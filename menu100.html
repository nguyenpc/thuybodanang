Danh Sách Thực Đơn Mức Giá 100.000 VNĐ

  • Trang Chủ
  • Thực Đơn

Set 1

  • 1) Khai vị:
    + Vả trộn tôm thịt
    + Bánh tráng nướng
  • 2) Ổ qua sốt patê
  • 3) Nghé xào lá lốt
  • 4) Bánh chưng dưa món
  • 5) Gà quay
  • 6) Đuôi bò hầm – bánh mì
  • 7) Tráng miệng: trái cây

Set 2

  • 1) Khai vị:
    + Bò bóp chuối sùng
  • 2) Súp cua nấm tuyết
  • 3) Mực hấp gừng
  • 4) Gà hấp giấy bạc
  • 5) Cơm dương châu
  • 6) Tim cật xào cải
  • 7) Tráng miệng: trái cây

Set 3

  • 1) Khai vị:
    + Sứa trộn
    + Bánh tráng nướng
  • 2) Tôm đất rang muối
  • 3) Gà hấp cải bẹ xanh
  • 4) Xôi dừa
  • 5) Ba chỉ giòn da
  • 6) Sườn nấu nấm đậu
  • 7) Tráng miệng: trái cây

Set 4

  • 1) Khai vị:
    + Gỏi bò ngũ sắc
    + Bánh tráng nướng
  • 2) Súp cua rong biển
  • 3) Tôm đất rang muối
  • 4) Gà hấp giấy bạc
  • 5) Mì quảng cá lóc (sườn nấu măng + bún)
  • 6) Tráng miệng: trái cây

Set 5

  • 1) Khai vị:
    + Sứa bóp chuối khế
    + Bánh tráng mè
  • 2) Ổ qua hầm patê (Súp lươn)
  • 3) Vịt quay củ kiệu
  • 4) Bánh bao chiên
  • 5) Xáo bò thập cẩm - Bún
