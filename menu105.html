Set 1

  • 2) Súp cua hải sản
  • 3) Gà hấp hành
  • 4) Xôi áp chảo
  • 5) Nghé xào lá lốt
  • 6) Sườn nấu măng – Bún ( Lẩu nấm sườn)
  • 7) Trái cây

Set 2

  • 1) Khai vị: Ổ qua sốt patê (Trứng vịt lộn um bầu)
  • 2) Gân bò bóp hành
  • 3) Cá diêu hồng chiên giòn cuốn lá cải
  • 4) Nghé xào lá lốt
  • 5) Lẫu hải sản – bún tươi
  • 6) Tráng miệng: trái cây

Set 3

  • 1) Khai vị: Gỏi ngũ sắc & Bánh phồng tôm

  • 3) Gà quay – Xôi chiên phồng
  • 4) Bê xào lá lốt
  • 5) Đuôi bò hầm sả - Bánh mì
  • 6) Tráng miệng: trái cây

Set 4

  • 7) Tráng miệng: trái cây

Set 5
